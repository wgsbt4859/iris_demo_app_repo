# アヤメの花の分類アプリ

教師あり学習の基礎で頻出の「アヤメの花の分類」について、分類予測を行うアプリを実装しました。(フレームワーク: streamlit)

## 分類モデル

初歩として簡易的なモデルとアプリを実装したかったため、ロジスティクス回帰モデルを用いています。
